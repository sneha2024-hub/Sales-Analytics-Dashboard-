# 📊 Global Retail Sales Performance & Customer Analytics Dashboard

An interactive Business Intelligence dashboard built using PostgreSQL and Power BI to analyze global retail sales performance, customer purchasing behaviour, and regional business trends.

The project transforms raw transactional retail data into actionable business insights through KPI reporting, customer analysis, sales trend visualization, and interactive dashboard storytelling. The dashboard enables decision-makers to monitor revenue performance, identify high-value customer segments, analyze regional sales distribution, and uncover product-level sales trends.

This project demonstrates end-to-end analytics workflows including SQL-based data extraction, data modeling, KPI analysis, DAX calculations, and dashboard development.

---

# 📌 Executive Summary

This project analyzes global retail transaction data to uncover business performance trends, customer purchasing patterns, and revenue-driving segments using PostgreSQL and Power BI.

The dashboard consolidates transactional sales data into an interactive reporting solution featuring KPI tracking, customer insights, regional sales analysis, and product performance visualization. Through SQL querying, data transformation, and Power BI dashboard development, the project converts raw retail data into clear, data-driven business insights.

The analysis identified that a relatively small group of high-value customers contributed significantly to total revenue, while sales performance varied across countries and product categories.

---

# 🎯 Business Problem

Retail businesses generate large volumes of transactional data daily, making it difficult to manually identify performance trends, customer behaviour patterns, and revenue opportunities.

Without centralized reporting and analytics systems, businesses may struggle to:
- Track revenue performance effectively
- Identify top-performing customer segments
- Monitor regional sales trends
- Analyze product-level sales distribution
- Support data-driven decision-making

The objective of this project was to build an interactive analytics dashboard capable of transforming raw retail sales data into structured insights for business performance monitoring and strategic analysis.

# 🛠 Methodology

## 🗄️ Data Extraction & SQL Analysis
- Imported and analyzed retail transaction data using PostgreSQL
- Performed SQL joins, aggregations, filtering, and data transformations
- Extracted customer, product, and sales-level insights

## 📊 Data Modeling & Dashboard Development
- Connected PostgreSQL data with Power BI
- Built relational data models for reporting
- Developed calculated measures and KPIs using DAX

## 📈 KPI & Business Analysis
Analyzed key business metrics including:
- Total Revenue
- Order Volume
- Product Quantity Sold
- Customer Contribution
- Regional Sales Distribution

## 🎨 Data Visualization
Created interactive dashboard visuals featuring:
- Revenue trend analysis
- Country-wise sales distribution
- Customer segmentation insights
- Product performance analysis
- Dynamic slicers and filters

# 📈 Key Insights & Results

- Identified that a relatively small segment of high-value customers contributed disproportionately to total revenue.
